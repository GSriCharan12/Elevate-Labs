A premium, interactive web application that fetches and displays user data from a public API using the JavaScript Fetch API.

## 🚀 Features
- **API Integration**: Fetches real-time user data from `JSONPlaceholder`.
- **Premium UI**: Modern dark-themed design with glassmorphism and animations.
- **Dynamic Rendering**: Automatically builds the user grid based on fetched data.
- **Error Handling**: Graceful handling of network errors with a retry option.
- **Responsive Design**: Fully functional across mobile, tablet, and desktop devices.
- **Micro-interactions**: Hover effects, staggered animations, and loading states.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom styling with CSS variables and animations.
- **JavaScript (ES6+)**: Fetch API for asynchronous requests and DOM manipulation.

## 📖 How to Use
1. Open `index.html` in your browser.
2. The application will automatically fetch data on load.
3. Click the **"Refetch Data"** button to refresh the list.
4. If the network is disconnected, an error modal will appear with a **"Try Again"** button.
