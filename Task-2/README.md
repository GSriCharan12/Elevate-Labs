# Task-2: Modern To-Do List Web App

A clean, responsive, and interactive To-Do List built with vanilla HTML, CSS, and JavaScript.

## Features
- **Add Tasks**: Quickly add new tasks with the input field or by pressing Enter.
- **Mark as Complete**: Click on any task text to toggle its completion status.
- **Delete Tasks**: Remove tasks with a smooth exit animation.
- **Empty State**: Visual feedback when the list is empty.
- **Data Persistence**: (Optional) tasks are currently stored in memory.

## How to Run
### Option 1: Open Directly
Simply double-click on `index.html` to open it in your default web browser.



## Structure
- `index.html`: The main structure of the application.
- `style.css`: Styling using modern CSS variables and animations.
- `script.js`: Logic for DOM manipulation and event handling.

## Technologies
- HTML5
- CSS3 (Flexbox, CSS Variables, Animations)
- Vanilla JavaScript (ES6+)
