# Task-8: Simple Blog Layout Using Bootstrap 5

A modern, responsive blog layout built with **Bootstrap 5**, featuring high-end aesthetics and premium UI components.

## 🚀 Features
- **Modern Branding**: Styled as **TECHNOVA** with a professional tech-focused color palette.
- **Responsive Grid**: Fully fluid layout that adapts to Mobile, Tablet, and Desktop screens.
- **Premium UI Components**:
  - **Magazine-style Featured Post**: A split-layout card for the latest articles.
  - **Interactive Blog Cards**: Smooth hover animations and glassmorphism effects.
  - **Sticky Navbar**: High-blur navigation for a modern feel.
  - **Sidebar Widgets**: Integrated search, categories, and a premium newsletter signup.
  - **Custom Pagination**: Clean, minimalist circular navigation.
- **Web Optimized**: Uses Google Fonts (Inter) and Bootstrap Icons for a crisp look.

## 🛠️ Built With
- **HTML5** & **Vanilla CSS**
- **Bootstrap 5.3.0** (CDN)
- **Google Fonts**
- **Bootstrap Icons**

## 📖 How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.

## 📁 Project Structure
- `index.html`: Main structure and Bootstrap components.
- `style.css`: Custom premium styling and animations.
