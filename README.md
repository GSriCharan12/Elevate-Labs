# Task-6: Premium Contact Form with Validation

A high-end, responsive contact form built with Vanilla HTML, CSS, and JavaScript. This project features robust client-side validation, anti-spam protection, and a cinematic dark-mode aesthetic.

## ‚ú® Features

- **Premium UI/UX**: Cinematic dark-mode design with glassmorphism, animated mesh gradients, and floating decorative blobs.
- **Robust Validation**: 
    - Real-time input validation for Name, Email, Phone, Subject, and Message.
    - Regex-based email and phone number formatting checks.
    - Minimum length requirements for specific fields.
- **Interactive Feedback**: 
    - Dynamic error messages with smooth slide-in animations.
    - Live character counter for the message field (`0/500`).
    - Visual focus states and button hover glow effects.
- **Anti-Spam**: Includes a hidden "honeypot" field to catch bots without affecting real users.
- **Responsive Design**: Optimized for seamless viewing on mobile, tablet, and desktop devices.
- **Modern Tech**: Uses Google Fonts (Outfit & Inter) and custom CSS cubic-bezier transitions.
