# Task-4: Mobile-Friendly Design with CSS Media Queries

## Objective
Convert an existing desktop-only luxury real estate landing page into a fully responsive, mobile-friendly website using CSS Media Queries.

## Features
- **Responsive Layout**: Fluid grids and containers that adapt from 4K displays down to small mobile screens.
- **Media Queries**: Custom breakpoints for tablets (768px) and mobile devices (480px).
- **Premium Aesthetics**: 
    - Glassmorphism effects on the navigation bar.
    - Smooth scrolling and micro-animations.
    - High-resolution architectural photography.
- **Mobile Navigation**: Reconfigured menu for vertical stacking and improved touch targets.
- **Optimized Typography**: Relative font sizing and scaled-down headlines for mobile readability.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Vanilla CSS, CSS Grid, Flexbox, Media Queries)
- Google Fonts (Outfit & Playfair Display)
